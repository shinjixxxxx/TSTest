# Claude Code、1年目に知っておきたかったこと 7選

**動画**: [YouTube で見る](https://www.youtube.com/watch?v=q-oA66ZUJJE&list=PL2noHKrggRBrJ4yUuvBigYeJX_9riVqtg&index=4)

Claude Code を活用して開発効率を劇的に高めるための 7 つの実践的なテクニック。
AIエンジニアを目指す方やプログラミング学習者に特に役立つ内容です。

---

## 開発効率を上げる 7 つのテクニック

1. **用途に応じて使い方を分ける** `0:25`
   初心者は環境構築のハードルが低い「デスクトップアプリ」から始めるのがおすすめ。

2. **3つのドキュメントを用意する** `1:39`
   `CLAUDE.md`（指示書）・`SPEC.md`（仕様書）・`TASKS.md`（タスク管理）を活用して、Claude にプロジェクトの文脈を理解させる。

3. **小分けにして実装させる** `5:29`
   一度に巨大なタスクを頼まず、タスクを分割して精度を維持する。
4. **「Planning → 実装」の順で進める** `6:40`
   いきなりコードを書かせず、まずはプランニングモードで設計を確認し、承認後に実装モードへ移行する。

5. **Claude に作業を検証させる** `8:09`
   成功基準（期待される動作やエラー時の挙動）を具体的に指示し、自律的な検証を促す。

6. **早期かつ頻繁に方向転換する** `9:08`
   意図と違う挙動を始めたら最後まで待たず、`ESC` キーで即座に中断して修正する。

7. **ショートカットキーを活用する** `10:03`
   `@`（ファイル参照）・`ESC`（中断）・`/compact`（会話の要約）などで効率アップ。
